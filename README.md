# **JH_Blog**

블로그 프로젝트입니다. 유저 관리, 게시글 등록, 실시간 채팅 기능 등을 우선적으로 설계하였고, 블로그라고 했지만 어울리는 모든 기능을 더해가며 역량을 키우는 것이 진행 목표입니다.


---

## **기능**

### 🔑 **유저 관련**
- **회원가입**
    - 로그인 아이디
    - 비밀번호
    - 프로필 사진
    - 닉네임
- **로그인**
    - 토큰 인증
    - OAuth 인증
- **로그아웃**
- **사용자 정보 수정**
- **다른 사용자 정보 확인**
- **회원 탈퇴**

---

### 📝 **포스트 관련**
- **게시글 등록**
    - 사진 등록
    - 제목 및 내용 작성
    - 댓글 작성

---

### 💬 **실시간 채팅**
- 원하는 사람과 실시간 채팅
    - 웹소켓(WebSocket) 사용

---

### ➕ **예상 업데이트 내역**
- 지역기반 정보 도입
    - 지역기반 정보를 사용한 편리한 기능 추가
- 취미가 맞는 그룹 생성
    - 취미가 아니더라도 공통 그룹을 생성 및 어울리는 기능 추가
